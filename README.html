Movie Trailer Website

This project is made by using python to create a static web page with our favourite movies trailers and information.

Required Libraries and Dependencies

Python 3.x is required to run this program.

Project Contents

The Project consists of following files:

  • media.py - contains the class movie that stores movie details
  • entertainment_center.py - main Python script to run
  • fresh_tomatoes.py - creates a static HTML file for the website

How to run

Open the text-based interface for your operating system (e.g. the terminal window in Linux, the command prompt in Windows). Navigate to the project directory and type in the following command:

python entertainment_center.py

Your default browser will open up the website.
